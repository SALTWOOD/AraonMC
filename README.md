# How to use

This application is not yet ready for use. Only a UI is available now.

# Development

## For IDE devs

1. Clone this repository.
2. Download and install [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0).
3. Open the solution file `AraonMC.slnx` (you can use Visual Studio or Rider to do this).
4. Build and run the project.

## For console devs

1. Clone this repository.
2. Download and install [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0).
3. Run `dotnet run --project AraonMC`
